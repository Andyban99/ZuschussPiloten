# Website deployen

Wenn Änderungen an der Website gemacht wurden, diesen Workflow ausführen.

## Schritte

// turbo-all
1. Zum Projektordner wechseln:
```bash
cd ~/Desktop/ZuschussPiloten
```

2. Alle Änderungen zum Commit hinzufügen:
```bash
git add .
```

3. Commit erstellen (BESCHREIBUNG anpassen!):
```bash
git commit -m "Website Update"
```

4. Zu GitHub pushen:
```bash
git push
```

Nach ca. 1 Minute ist die Website unter https://andyban99.github.io/ZuschussPiloten/ (und bald https://zuschusspiloten.de) aktualisiert.
